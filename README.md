## ✏ EduWithMe
EduWithMe는 모든 연령과 배경의 사용자들이 언제 어디서나 접근할 수 있는 개인 맞춤형 학습을 통해 성장하며, 선생님과 학생 간의 상호작용을 강화하고 학습 동기를 높이는 교육 플랫폼입니다.

<br>

## 📌 주요 기능
- 방 생성 및 문제 출제
- 포인트 획득 및 랭킹 시스템
- 마이페이지 기능
- 채팅 기능
- AI 질문하기
